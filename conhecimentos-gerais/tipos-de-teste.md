# Tipos de Teste de Software

## Funcionais
- Verificam se as funções do software trabalham conforme o esperado.

## Não Funcionais
- Testam aspectos como desempenho, usabilidade e segurança.

## Exemplos:
- Teste de Regressão
- Teste de Integração
- Teste de Usabilidade
- Teste de Performance
- Teste de Segurança
